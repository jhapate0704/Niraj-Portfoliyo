Smart Bookmark


Engineered a premium, highly interactive Google Chrome Extension that modernizes the browser's bookmarking system by transforming it into an intuitive, native iOS 26/VisionOS inspired environment.The application focuses heavily on high-performance animations, intelligent categorization, and advanced glassmorphism design.

Browser API Integration: Deeply integrated with Chrome's native capabilities, utilizing the Context Menu API to allow one-click AI-powered bookmark tagging directly from web pages, and the Notifications API for seamless user feedback.

It solves a real problem: Bookmarks are boring. You built a tool that actually improves the browser experience, which is exactly what a product team looks for.

Ai-Pull-Requests-Reviewer


Developed an intelligent, full-stack application that acts as an automated CI/CD code reviewer. By leveraging advanced Large Language Models (LLaMA 3.3 via Groq), the system automatically fetches GitHub Pull Requests, analyzes code diffs file-by-file, and identifies bugs,security vulnerabilities, and code quality issues—providing actionable fixes directly as GitHub comments.

Zero-Click Automation & Webhooks: Engineered a background task processing system that listens to GitHub pull_request webhook events, automatically triggering AI inference and posting contextual markdown reviews directly to the PR without user intervention.

Detailed File-by-File Review: Processes each file's diff individually rather than merging them, ensuring context is accurately maintained, hallucination is minimized, and feedback is highly targeted.

Direct Fix Suggestions: AI provides actionable code snippets and suggestions (💡 Fix: ...) for developers to quickly resolve issues.
